# Field Tested
Rowan University Spring 2026 Computing and Informatics Capstone Project

## About
Field Tested is a 2D sidescroller platformer built with Love2D and Lua. 
The player must collect keys to advance through levels.

## How to Run & Requirements
Love2D version 11.4 or higher
Windows 7+ or MacOS 10.11+

### Installation
1. Download and install Love2D from https://love2d.org
2. Download or clone this repository
3. Navigate to the 'Field Tested' folder

### Running on Windows & MAC
Drag and drop the 'Field Tested' folder onto 'love.exe'

## Controls
- A / Left Arrow — Move left
- D / Right Arrow — Move right
- W / Space / Up Arrow — Jump
- E — Interact with objects and signs
- R — Reset to spawn point
- ESC — Pause game

## Features
- 2D sidescroller platformer
- Puzzle mechanics with interactive UI
- Orb collection system
- Local save and load system
- Animated player character
- Original music and sound effects

## Libraries Used
- Love2D — Game engine
- anim8 — Animation
- STI (Simple Tiled Implementation) — Map loading
- HUMP — Gamestate management
- Camera — Camera system
- Bitser — Save file serialization

## Tools Used
- Tiled — Map editor
- Piskel — Sprite and animation design
- Visual Studio Code — IDE
- GitHub — Version control
